# React Auth Pages

Single-page React UI demonstrating a login form and a register form with modern styling,
responsive layout, and inline validation.

## Getting started

```bash
npm install
npm start
```

The development server runs on [http://localhost:3000](http://localhost:3000) by default.

## Features

- Create React App foundation inside `frontend/`
- Shared design system with gradients, glassy cards, and accessible controls
- Client-side validation for both login and register flows
- Toggle between login and register without navigating away

## Production build

```bash
npm run build
```

The optimized production output resides under `frontend/build`.
